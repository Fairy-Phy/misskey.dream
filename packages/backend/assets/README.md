# Misskey.Dream アイコンについて

Misskey.Dreamアイコンは以下のファイルのアイコンのことを指します。

* icons/192.png
* icons/512.png
* apple-touch-icon.png
* favicon.ico
* favicon.png
* splash.png
* long-logo.png

これらアイコンはMisskey.Dreamによって作成されたものです。
